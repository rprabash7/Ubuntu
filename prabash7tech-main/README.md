# Prabash7Tech - Portfolio Website

Django + React Full Stack Portfolio Website for programming tutorials in Telugu.

## Tech Stack

- **Backend**: Python Django + Django REST Framework
- **Frontend**: React.js with Tailwind CSS
- **Database**: SQLite (Development)

## Features

- Responsive Design
- Django REST API
- React Components
- Image Gallery
- Blog Section
- Contact Information

## Local Development

### Backend Setup
```bash
python3 -m venv env
source env/bin/activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver

cd frontend
npm install
npm run build

Deployment

Ready for EC2 deployment with Nginx and Gunicorn.
